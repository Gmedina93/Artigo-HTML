O que é HTML

HTMLé uma linguagem de marcação de hipertexto utilizada para criar páginas web. É interpretado por navegadores(Chrome, Mozilla, Internet explorer)

As marcações são referenciadas como tags e a várias tags cada uma com sua utilidade.

Tags

Tags são conjuntos de caracteres que formam um elemento, de modo prático as tags são feitas assim:

Tag para criar um parágrafo: < p> < /p>

Existem dois tipos de tags, as que precisamos fazer o fechamento como no exemplo anterior e as que não precisam de fechamento.

Exemplos de Tags

Vou listar alguns exemplos de tags que podemos usar no nosso projeto HTML

  • < h1> Tag de título < /h1>

    Serve para formatar o texto entre as tags como titulo:

    Exemplo:

    Titulo

    Alterando a numeração podemos ter outros valores, é bem utilizado para titulos e subtitulos

  • < u> Tag de sublinhado< /u>

    Serve para sublinhar todo o texto marcado.

    Exemplo: Sublinhado
